# 🤖 Machine Learning Learning Journey

This repository contains my implementations of core machine learning concepts inspired by lectures and tutorials from Andrej Karpathy, Andrew Ng, CS4780 and Deep learning - Christopher Bishop

---

## 📚 Projects

### 🔹 Backpropagation from Scratch
Implemented automatic differentiation and neural network training using a micrograd-style engine.

### 🔹 Character-level Language Model (Bigram)
Built a probabilistic model to generate names using bigram statistics.

### 🔹 MLP Language Model with Context
Extended the bigram model using embeddings and a neural network.
